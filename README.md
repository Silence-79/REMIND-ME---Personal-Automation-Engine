# REMIND US

A modern, classy automation and reminder platform built with React.

## Features

### 🎯 Smart Automation Rules
- Create custom automation rules with multiple trigger types (Button, Interval, Time)
- Set priority levels (High, Medium, Low) for rule execution
- Enable/disable rules on the fly
- Test rules manually before scheduling

### 🔍 Advanced Filtering
- Search rules by message content
- Filter by trigger type (Button, Interval, Time)
- Filter by status (Enabled/Disabled)
- Real-time filtering with instant results

### 📊 Dashboard Analytics
- View total rules count
- Track active vs disabled rules
- Clean, visual statistics display

### 📝 Execution Logging
- Automatic logging of all rule executions
- Timestamp tracking for each execution
- Trigger type identification
- Manual log clearing option

### 💾 Import/Export
- Export all rules as JSON
- Import rules from JSON files
- Backup and restore functionality
- Share rule configurations

### 🎨 Modern UI
- Animated gradient header
- Glass morphism design
- Smooth transitions and hover effects
- Responsive layout for all devices
- Dark theme with elegant color palette

## Tech Stack

- React 19
- Vite
- LocalStorage for persistence
- CSS3 with animations

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## Usage

1. **Create a Rule**: Choose a trigger type, set priority, and add your message
2. **Manage Rules**: Enable/disable, edit, test, or delete rules
3. **Filter & Search**: Find specific rules using the filter bar
4. **Monitor Execution**: View execution history in the log viewer
5. **Backup**: Export your rules for safekeeping or sharing
